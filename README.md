# practicaFinalSO

Proyecto realizado en JAVA

Cuando ejecutas el te pedirá ingresar cadena (Enter String), por ejemplo:
-> 2+3+4+5-6-7
-> 3-5+4-2*2
-> 3*4*2+2
-> 9/3+4
-> 6/2-3+6

Cadenas NO! validas:
-> 4+-5+4
-> a*b-4
-> 9/0-1
-> *4+4-4-
-> S-A+N*t.1
ETC....

El programa está hecho utilizando conceptos trabajados en la materia Compiladores.
